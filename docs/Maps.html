Web Map Samples

These maps were first created as part of a JS web mapping course,
but include a few more details and options than were neccesary because it's fun to play with them

Map Of Public Transport Average Time Varition From Schedule

The map was created based on data collected for my thesis. in the city of Be'er Sheva, Israel, for a week in July 2017. the data presented respresnts the mean time variation between real time bus GPS data collected every 30 seconds and the bus schedule.

Map Of Rahat, Israel Lots

Started as a pet project to understand what are the possibilities of web maps, i am still exploring and adding more features when i can think of them and implement them. the final goal here is to create an open-source map a municipality can use.
Current Implemented features are:

  • ☑ Multiple Basemaps Available
  • ☑ Address Search Using OSM data
  • ☑ Measurement of Length and Area of user drawn elements
  • ☑ Cursor Coordinates Location displayed
Features I still want to implement:
  • ☐ Selection from data presented, single and multiple objects
  • ☐ Export of data selected
  • ☐ Ability to create polygons and
    • ☐ Suggest they be inserted to the database
    • ☐ Export them as a GeoJSON file
  • ☐ Last clicked location observer
    • ☑ Coordinates
    • ☐ Closest neighborhood and lot
  • ☐ Creation of Buffer and selection by buffers
